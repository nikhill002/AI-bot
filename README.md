# NikkaBot

NikkaBot is a lightweight, personalized AI assistant built with Python and powered by Llama 3 via Ollama. 

## Description
Designed to run locally on your machine, it provides a fast, private, and efficient interaction experience. With a custom system prompt that defines a calm, professional, and slightly witty persona, NikkaBot is tailored to be a helpful assistant for daily tasks while keeping your data strictly on your local device. MAKE SURE YOU HAVE OLLAMA INSTALLED ON YOUR SYSTEM OTHERWISE THIS PROJECT WON'T WORK.

## Key Features
* **Local-First Architecture:** Runs entirely on your hardware using Ollama; no data leaves your machine.
* **Custom Persona:** Engineered with a specific system profile to ensure helpful, concise, and professional responses.
* **Lightweight:** Built using Python, making it easy to maintain and expand.
* **Private:** No API keys or cloud dependencies required.
